# NFL Division Quiz

A React web app to help you learn which teams belong to each NFL division.

## Game Modes

- **Guess the Division**: See a random NFL team and guess which division it belongs to
- **Assign Teams**: Drag and drop all 32 NFL teams into their correct divisions

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server with hot reload |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |

## Tech Stack

- React 19
- Vite 7
